# SmartPath - A Degrere Planning tool

### Project Description
SmartPath was envisioned as a web app that allows students to plan their degree as well as share those plans with advisors and other users. It also allows for advisors to give feedback on plans
This project was created as coursework for our software engineering course. 

Prequisites: 
- NodeJS and npm, which you can install from on Node's [website](https://nodejs.org/en/download)

How to Run:
- In your terminal, run `node server.js` in the server directory and `npm start` in the inner smartpath directory.
- A browser window should open with the app, or you can navigate to [localhost:3000](https://localhost:3000).
